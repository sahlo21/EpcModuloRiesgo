Artifact: Informe de proyecto
El informe del proyecto se mencionan posibles cambios o mejoras respecto al resultado de la consulta de experiencia.
Purpose
El proposito de este artefacto es tener una guia de cambios a realizar dentro de la apliacion.
Relationships
RolesResponsible:
  • Gerente del proyecto
Modified By:
  • Gerente del proyecto
TasksInput To:
  • Realizar documento con mejoras y cambios
Output From:
  • Analizar la grafica con los resultados de los formularios
  • Realizar documento con mejoras y cambios
Description
Main Description

El informe del proyecto se construye a partir de informacion y decisiones que se tomaron en la actividad de consulta de experiencia, en este se evidencia la informacion de posibles cambios o mejoras que se le pueden realizar a la apliacion.
