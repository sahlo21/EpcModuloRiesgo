Role: Tester QA
El tester QA es el rol encargado de realizar pruebas a las aplicaciones realizadas por los desarrolladores.
Relationships
Primary Performs
  • Llenar formulario de experiencia
  • Testear objeto ejecutable personalizado y parametrizado
Additionally Performs
  • Analizar la grafica con los resultados de los formularios
  • Realizar documento con mejoras y cambios
Modifies
  • Grafica de formulario
    Main Description
    El rol de Tester QA dentro de un equipo de trabajo es aquel que se encarga de realizar pruebas funcionales y no funcionales al programa a desarrollar dentro del proyecto, todo esto con el fin de encontrar errores y realizar la notificacion de estos.
    Staffing
    Skills

    El tester QA debe tener como habilidades el analisis y la intuicion con el fin de saber que pruebas realizar a que programas y saber en que lugares es mas probables que se genere un error.

    Assignment ApproachesEl enfoque principal de los Testers QA es realizar pruebas funcionales y no funcionales a los programas, todo esto con el fin de encontrar problemas para que los desarrolladores den soluciones.
