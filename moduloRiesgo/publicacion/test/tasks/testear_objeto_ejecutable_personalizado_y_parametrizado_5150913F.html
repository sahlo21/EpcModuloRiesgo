Task: Testear objeto ejecutable personalizado y parametrizado
Esta tarea busca realizar pruebas a el proyecto con un prototipo y a un grupo de usuarios.
Purpose
El proposito de esta tarea es obtener el nivel de aceptacion y usabilidad de un prototipo de la aplicacion software.

    Main Description

    La tarea de testear un objeto ejecutable personalizado y parametrizado se basa en realizar una prueba a un grupo de usuarios seleccionados, con el fin de obtener un feedback y de esta manera realizar cambios, mejoras y correcciones al producto software de nuestro proyecto.

    Steps
    Realizar el prototipo del sistema
    Se realiza un ejecutable a modo de prototipo de nuestro producto software, este cumple con la funcion principal pero no sera el que se suba a produccion.
    Seleccionar los usuarios
    Luego de tener el prototipo se buscan usuarios que puedan realizar la prueba y que cumplan con los requisitos para estas.
    Realizar la prueba
    Se realiza la prueba con los usuarios y se obtiene el feedback de estos.
