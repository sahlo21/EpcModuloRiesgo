Main Description
El modulo de gestion de riesgo dentro de la empresa constructora tiene diversas funciones, en las cuales se destaca la notificacion a los demas modulos acerca de posibles riesgo que se van a dar dentro de esta, ademas de esta funcion tambien puede habilitar o deshabilitar empleados, sin embargo, en esta pagina web se busca mostrar a las personas el proceso que se realiza para obtener un proyecto gestionado correctamente, para esto se muestran los roles, tareas y artefactos que se realizarob, asi como su orden y diagramas correspondientes.
