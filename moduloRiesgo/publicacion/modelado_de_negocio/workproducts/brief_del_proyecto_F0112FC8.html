Artifact: Brief del proyecto
El brief del proyecto es el documento en el cual se tiene que personas participaran en el proyecto, asi mismo las funciones de estos dentro del proyecto y las tareas que van a realizar
Purpose
El proposito del brief del proyecto es generar informacion con los stakeholders del proyecto.
Relationships
RolesResponsible:
  • Gerente del proyecto
Modified By:
  • Gerente del proyecto
TasksInput To:
  • Asignar las funciones
  • Revisar funciones y equipo
Output From:
  • Asignar las funciones
  • Investigar el equipo
Description
Main Description

El artefacto o documento brief del proyecto se realiza con el fin de tener clara la informacion de los interesados del proyecto, tanto de manera externa como lo son los usuarios, como de manera interna como lo son los miembros del equipo de desarrollo del proyecto.
