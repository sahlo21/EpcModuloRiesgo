Artifact: Documento del proyecto
El documento del proyecto posee como informacion la arquitectura de software.
Purpose
El proposito del documento del proyecto es tener una guia detallada del proyecto a desarrollar.

Description
Main DescriptionEl documento del proyecto tiene en su interior toda la arquitectura del software, es decir, tiene los diagramas de caso de uso, el diagrama de parapeto, las reglas de negocio, presupuesto, requisitos, entre otros... Este documento tiene como objetivo dejar claro el alcance y manera de funcionamiento del proyecto.
