Task: Investigar el equipo
La tarea de investigar el equipo cumple la funcion de obtener de mejor manera las habilidades y conocimientos que tienen los participantes del proyecto, esto con el fin de asignar de mejor manera las tareas y roles
Purpose
El proposito de esta tarea es obtener la informacion de las personas que hacen parte del equipo del proyecto.

    Main Description

    La tarea de investigar el equipo de trabajo se realiza con el fin de obtener diversos datos de las personas que conforman el equipo, entre estos datos se tiene las habilidades que poseen, los roles que suelen desempe√±ar y la manera en la que trabajan en equipos.

    Steps
    Obtencion del equipo

    Para el proyecto se busca un equipo que este capacitado en las necesidades de los stakeholders, para esto se realiza una investigacion sobre los equipos y se selecciona el mas capacitado para el proyecto.

    Contratacion del equipo
    Luego de obtener el equipo se realiza una contratacion, en esta se analizan los salarios, horas y las leyes vigentes para la realizacion del contrato.
