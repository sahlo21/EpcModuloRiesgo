Role: Arquitecto
El arquitecto del proyecto es el encargado de realizar la documentacion pertinente del proyecto.
Relationships
Additionally Performs
  • Iniciar el proyecto
  • Revisar funciones y equipo
Main Description
El arquitecto del proyecto es el rol encargado de realizar la base del proyecto de software, es decir, se encarga de realizar toda la documentacion/arquitectura del proyecto a realizar, ademas de esto, se encarga de realizar los requisitos los cuales son un de los pilares fundamentales del proyecto.
Staffing
Skills

El arquitecto de software debe tener una comunicacion acertiva con todo el equipo, ademas de una capacidad de analisis bastante desarrollada con el fin de obtener la informacion de la mejor manera.

Assignment ApproachesEl arquitecto de software se enfoca en la creacion y desarrollo de la documentacion del software, sin embargo tambien puede elegir las tecnologias a usar en el proyecto.
Key Considerations
El arquitecto esta en constante comunicacion con el equipo de desarrolladores con el fin de evitar inconvenientes.
